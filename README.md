# JobTrackIQ

JobTrackIQ is a data-driven job application tracking and recommendation system.

## Current Features (MVP)
- Track job applications and application events
- Maintain a unified job post pool (manual import)
- Compute application funnel metrics

## Tech Stack
- FastAPI, SQLAlchemy
- PostgreSQL / SQLite
- Python (ML & analytics), R (reporting)

## Roadmap
- Job matching & recommendations
- Success probability prediction
- Semi-automated application workflow
